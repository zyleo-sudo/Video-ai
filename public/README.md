# Video AI - 快速开始

## 🚀 启动应用

### Windows 用户
双击运行 `start-server.bat`

### Mac/Linux 用户
在终端运行：
```bash
chmod +x start-server.sh && ./start-server.sh
```

⚠️ **重要**：请确保在解压后的目录中运行启动脚本（不要将脚本复制到其他位置）

然后在浏览器访问：http://localhost:3000

---

## 🔑 设置 API 密钥

1. 点击右上角「设置密钥」按钮
2. 访问 [allapi.store](https://allapi.store) 获取 API 密钥
3. 输入密钥并保存

## ⚠️ 注意事项

- 首次使用前请确保已安装 Node.js
- API 密钥存储在您的浏览器本地，不会泄露
- 按 Ctrl+C 可停止服务器
- 建议使用 Chrome 或 Edge 浏览器
