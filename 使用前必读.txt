# AI 视频创作工具 (Web 版) - 全免费/零门槛运行指南

这是一个已经过优化的全自动运行版本。即使朋友的电脑没有安装任何开发环境，也可以通过以下步骤直接运行。

## 🚀 启动步骤

1. **解压文件夹**：将收到的压缩包解压到一个文件夹中。
2. **免安装启动**：
   - **双击 `启动程序.bat`**。
   - **完全自动化**：脚本会自动检测电脑环境。如果没有 Node.js，它会自动从官网下载一个轻量化的集成环境并完成配置，无需您手动安装任何东西。
   - **自动打开**：环境准备好后，它会自动启动应用并弹出您的浏览器窗口。

3. **填入密钥**：
   - 进入应用后，点击左侧边栏的 **⚙️ 设置 (Settings)**。
   - 填入您自己的 API Key 和 中转地址，即可开始生成。

---

## 🛠 给分享者的建议 (打包前必读)

为了让朋友用得更爽，您在发给他之前可以做以下操作：

- **缩小体积**：您可以删除 `node_modules` 文件夹和 `.node` 文件夹（如果存在）。这些文件夹在朋友第一次运行时会自动重新下载。
- **确认配置**：请放心，这个脚本**绝不会**包含您的 API 密钥，该密钥仅存在于您本地浏览器的缓存中。
